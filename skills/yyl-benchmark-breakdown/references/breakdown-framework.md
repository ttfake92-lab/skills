# 拆解框架:方法论 + 各平台维度 + 爆款公式提炼

原则:**别复述对方说了什么,要说清"为什么这么做"和"我怎么套用"。** 每个结论回到「戳了观众什么情绪 / 解决了什么问题」。

---

## A. 通用四层拆解(任何平台都过一遍)

1. **选题层**:这条切的是什么需求/情绪?(实用干货 / 情绪共鸣 / 猎奇 / 身份认同 / 焦虑或爽感)。为什么这个选题能让人停下来?
2. **结构层**:整体怎么编排的?(钩子 → 铺垫 → 主体 → 高潮 → 收尾/CTA)。每段在干什么、时长/字数占比。
3. **表达层(文字/口播侧)**:钩子怎么写的、节奏快慢、有没有反转、金句、口头禅、标题套路。
4. **表达层(画面侧)**:视觉钩子、镜头景别、切点节奏、字幕样式、视觉符号、色调、场景调度。**详见 `visual-framework.md`,短视频拆解必须过一遍。**
5. **转化层**:想让观众做什么?(关注、点赞、收藏、评论、跳转、下单)。CTA 怎么埋的、互动钩子(提问/争议/留悬念)在哪。

---

## B. 各平台特化维度(在四层之上叠加)

### 抖音 / 视频号(短视频,黄金 3 秒)
- **前 3 秒钩子**逐帧/逐句拆:画面 + 第一句话 + 字幕,怎么制造「不划走」。
- 节奏:多少秒一个信息点 / 反转;有没有卡点、音乐情绪。
- 完播设计:有没有「留到最后」的悬念、清单式结构。
- 评论区运营:作者置顶/回复怎么引导二次传播。

### 小红书(图文/视频笔记,封面+标题决定点击)
- **封面**:第一眼信息、文字排版、对比/痛点呈现。
- **标题**:关键词(搜索)、情绪词、数字、利益点;20字内怎么塞钩子。
- 正文结构:开头痛点 → 干货分点 → 结尾互动;emoji/分段/小标题节奏。
- 标签 + 关键词:蹭了哪些流量词;种草/转化埋点。

### B站(中长视频,重选题与留存)
- 标题 + 封面的「点进来」组合;选题的稀缺性/信息增量。
- 开头 30 秒留存设计(预告价值、抛悬念)。
- 信息结构:章节、节奏起伏、高能点分布;三连引导话术。

### YouTube(同 B站 逻辑,叠加)
- Thumbnail + Title 的好奇缺口(curiosity gap)。
- 开场 hook 与 retention 设计;packaging(选题包装)。

### 公众号 / 图文长文
- 标题党手法(悬念/反差/利益/身份);开头怎么抓住继续读。
- 叙事节奏:故事 vs 干货 vs 观点;金句与转发点(可截图传播的句子)。
- 转化:文末引导、引流钩子、留言区运营。

## C. 账号级拆解(链接是主页时)

抓 5–15 条样本,找**重复出现的规律**,而不是逐条讲:
- **选题矩阵**:内容分成哪几类?哪类占比高、哪类数据最好?
- **爆款共性**:数据明显高的几条,共同点是什么(选题/结构/标题/封面套路)?
- **人设标签**:对方给自己贴了什么标签?口头禅、视觉符号、价值主张。
- **更新规律**:发布频率、时段、形式偏好。
- **差异化**:相比同赛道,它强在哪、留了什么空位(我能切入的角度)。

## D. 爆款公式提炼法(三件套之「公式」的产出方法)

把观察**升维成能套用的模板**,例:

- **选题公式**:`<目标人群> + <强情绪/痛点> + <反常识或利益点>`
  例:「普通人也能…的 3 个方法」=身份认同 + 低门槛 + 数字利益。
- **结构公式**:写成可填空的骨架。
  例:`痛点暴击(3s) → 我也踩过坑(共情) → 方法1/2/3(干货) → 反转金句 → 关注引导`
- **标题/封面公式**:抽出可复用的句式与视觉套路。

每个公式后面跟一句:**「套用到我的选题会是____」** —— 直接给一个改编示例,让产出可立即用。
