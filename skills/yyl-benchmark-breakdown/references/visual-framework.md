# 视觉拆解框架:画面是脚本的另一半

短视频拆解只看口播会漏掉一半「为什么火」。剧情片、Vlog、美食、穿搭、无口播剪辑这些类型,**画面承担的信息量 ≥ 口播**。前 3 秒的视觉钩子常常比第一句话更决定划走与否。

---

## 0. 抽帧 + Contact Sheet(动手前先有素材)

详见 `fetch-playbook.md` 第 3 节(命令模板的权威版本)。简化要点:

```bash
mkdir -p /tmp/bm/frames

# a) 场景检测找切点(阈值 0.3 一般合适,慢节奏可降到 0.2)
ffmpeg -i /tmp/bm/video.mp4 -vf "select='gt(scene,0.3)',showinfo" \
  -fps_mode vfr /tmp/bm/frames/cut_%03d.jpg 2>/tmp/bm/cuts.log

# b) 前 3 秒每秒一帧(锁住钩子)
ffmpeg -i /tmp/bm/video.mp4 -t 3 -vf "fps=1" -fps_mode vfr /tmp/bm/frames/hook_%03d.jpg -loglevel error

# c) 整体节奏补抽(每 3 秒一帧,<60s 短视频改 fps=1/2)
ffmpeg -i /tmp/bm/video.mp4 -vf "fps=1/3" -fps_mode vfr /tmp/bm/frames/t_%03d.jpg -loglevel error

# d) 拼 sheet(6x6=36 张/页,超过自动分多张)
ffmpeg -pattern_type glob -i "/tmp/bm/frames/*.jpg" \
  -filter_complex "scale=320:-1,tile=6x6:padding=4:color=white" \
  -fps_mode vfr /tmp/bm/sheet_%03d.jpg -loglevel error

# e) 剪辑节奏数字
TOTAL_CUTS=$(grep -c "pts_time" /tmp/bm/cuts.log)
DURATION=$(ffprobe -v error -show_entries format=duration -of csv=p=0 /tmp/bm/video.mp4)
awk -v d="$DURATION" -v c="$TOTAL_CUTS" 'BEGIN { printf "切点 %d / 时长 %.1fs / 平均 %.2fs 一切\n", c, d, d/c }'
```

产出:
- `/tmp/bm/sheet_*.jpg` —— **逐张 Read,多模态读图**(可能多张,都要读)
- `/tmp/bm/cuts.log` —— 切点时间戳(算剪辑节奏 / 对齐口播段落)
- `/tmp/bm/frames/cut_*.jpg`、`hook_*.jpg` —— 需要细看某帧时单独 Read

**长视频(>5 分钟)**:把 `fps=1/3` 改成 `fps=1/5` 或 `fps=1/10` 降密度,sheet 会自动分多张。

## 1. 七个视觉拆解维度

### ① 前 3 秒视觉钩子(短视频最关键)
- 第一帧画面是什么?**人脸特写 / 行动中 / 异常物体 / 反差场景 / 文字大字报**?
- 前 3 秒切了几次?(1 次=稳镜抓人,3+ 次=信息密度轰炸)
- 有无大字幕/文字卡?写了什么?字号占画面比例?
- 钩子类型归类:**悬念型 / 反差型 / 利益前置型 / 情绪冲击型 / 身份认同型**

### ② 镜头景别分布
- 特写 / 中景 / 全景 / 远景 各自比例。
- **特写多** = 情绪带入强(适合人物/剧情/口播带情绪)
- **中景多** = 信息传达稳(适合教程/讲解/Vlog)
- **远景多** = 场景叙事(适合旅拍/纪录/剧情)
- 切换有没有规律(eg. 每个干货点开头都用特写)

### ③ 切点节奏 = 剪辑密度
- 平均多少秒一切?(参考:抖音热门 1-3 秒,Vlog 3-5 秒,B站知识区 5-10 秒)
- 节奏是均匀的还是「快-慢-快」(钩子密 → 主体放 → 结尾再密)?
- 有没有卡点(切点对齐音乐节拍)?

### ④ 字幕/文字信息卡(屏幕文字)
- 字幕样式:**位置(顶/中/底) / 字号 / 颜色 / 描边 / 出现方式(逐字弹/整句切)**。
- 字幕 ≠ 口播逐字稿:有没有**关键词加色、强调字放大、和口播错位**(口播说A字幕写A+1)?
- 信息卡:**标题卡 / 章节卡 / 数据卡 / 弹幕式吐槽** —— 在什么时机出现,起什么作用?

### ⑤ 视觉符号 / 记忆点(账号级看)
- 反复出现的元素:**固定开场动作 / 标志服装 / 标志道具 / 标志场景 / 标志手势**。
- 这是账号「视觉品牌」,是观众**0.5 秒认出来**的核心。
- 对单条:有没有用上账号既有的视觉符号?

### ⑥ 色调与光线
- 整体色调:**暖橘 / 冷蓝 / 高饱和 / 低饱和电影感 / 滤镜风格**。
- 光线:**自然光 / 棚拍硬光 / 柔光 / 暗调氛围**。
- 跟选题/情绪匹配吗?(eg. 治愈系用暖橘+柔光,科技测评用冷调+硬光)

### ⑦ 场景调度
- 单一场景 vs 多场景?多场景之间怎么转(硬切 / 同位置同人物变状态 / 镜头匹配剪辑)?
- 有没有用「场景变化」推动叙事节奏(eg. 室内 → 户外 = 情绪转折)?

## 2. 各平台视觉特化

### 抖音 / 视频号
- **9:16 竖屏构图**:主体在不在安全区(顶部 1/5 留给标题、底部 1/5 留给字幕和UI)?
- 前 3 秒视觉钩子权重 > 口播。
- 字幕几乎全屏跟读;有无关键词放大/变色。
- 卡点剪辑(对齐背景音乐节拍)是不是普遍?

### 小红书(视频)
- **3:4 / 1:1 构图**,封面是第一帧还是单独设计?
- 封面文字与视频内容的「点击-内容」一致性。
- 干货类常用「分屏 / 对比 / 步骤标号」。

### 小红书(图文)—— 没视频,看图序
- 第一张图(封面)信息密度 = 标题 + 痛点呈现 + 视觉钩子。
- 后续图序逻辑:**问题 → 干货 → 效果** / **对比前后** / **步骤 1234**。
- 文字在图上的排版风格(手写感 / 极简衬线 / 综艺花字)。

### B站 / YouTube
- **16:9 横屏**,封面+标题组合的好奇缺口。
- 开场 30 秒视觉:**预告精彩片段 / 抛悬念道具 / 第一人称带入场景**?
- 中长视频看「视觉节奏起伏」:静态讲解 vs 实拍演示 vs 动效图表 各自占比。

### 公众号 / 图文长文
- 配图风格(摄影 / 插画 / 截图 / 数据图)与正文调性一致?
- 关键金句有没有做成卡片图便于截图传播?

## 3. 视觉 + 口播 时间轴对齐(脚本拆解升级)

光列视觉维度不够,真正的拆解是**让画面和口播在时间轴上对齐**,看两者怎么协同/冲突/补强:

| 时间 | 画面(从 sheet/cuts) | 口播(从 whisper) | 协同方式 |
|---|---|---|---|
| 0-3s | 特写 + 红字"她疯了吗" | "我跟你说一个事..." | 视觉抛悬念 + 口播留扣子 |
| 3-8s | 中景人物走动 | "事情是这样的..." | 视觉降密度让口播信息进场 |
| ... | | | |

**协同方式**类型:
- **同步**:画面 + 口播说一件事(信息冗余强调)
- **互补**:画面给情境,口播给观点(信息分工)
- **错位**:画面演 A,口播说 B(制造冲突/反差/幽默)
- **预告**:画面先剧透下一段口播(留悬念)
- **强调**:口播说关键词的瞬间画面突然给特写/字幕放大

## 4. 视觉公式怎么提炼(产出三件套之一)

跟选题/结构公式一样,把观察升维成模板:

- **视觉钩子公式**:`<景别> + <反差元素> + <屏幕大字> = 0.5 秒留住人`
  例:特写人脸 + 异常表情 + "千万别这样!" 红字
- **剪辑节奏公式**:`钩子段 X 秒一切(密) → 主体段 Y 秒一切(放) → 结尾段 Z 秒一切(再密)`
- **视觉符号公式**:抽出**可立刻复用**的画面元素(eg. 固定开场手势、固定场景、固定字幕字体)

每个公式后面跟一句:**「套用到我就是____」** —— 给出具体画面建议,不要空泛。
