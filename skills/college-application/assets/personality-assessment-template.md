# 高考志愿决策辅助：性格与职业兴趣测评表

> 用途：为高考志愿填报提供“职业兴趣 + 人格倾向”的初步线索。它不是心理诊断，也不能单独决定专业选择。最终结论必须结合真实经历、职业/行业研究、高考成绩/位次、招生规则和家庭现实约束。

## 科学依据与来源

本表采用公开、可引用的非商业框架：

1. **RIASEC / Holland 职业兴趣类型论**：用于识别职业兴趣环境倾向（Realistic、Investigative、Artistic、Social、Enterprising、Conventional）。职业映射可参考 O*NET OnLine（美国劳工部职业信息系统）：https://www.onetonline.org/
2. **Mini-IPIP / Big Five 短量表**：用于识别大五人格倾向。量表论文 DOI：https://doi.org/10.1037/1040-3590.18.2.192；IPIP Mini-IPIP 计分页：https://ipip.ori.org/MiniIPIPKey.htm
3. **专业映射**：大学专业名称和分类必须回到教育部《普通高等学校本科专业目录》或高校官方培养方案核验；不要只凭职业名称推专业。

## 填写说明

请按第一直觉打分：

- 1 = 完全不符合（偏“否”）
- 2 = 比较不符合
- 3 = 说不清 / 一般
- 4 = 比较符合
- 5 = 非常符合（偏“是”）

填完后保存这份文档，并把文件发回、粘贴文件内容，或授权 Agent 读取该文件。不要在终端里逐题回复；终端只用于告诉 Agent 文档已经填好。

## 计分规则

### RIASEC

- R = A01 + A02
- I = A03 + A04
- A = A05 + A06
- S = A07 + A08
- E = A09 + A10
- C = A11 + A12

每项范围 2–10。取前三名形成 RIASEC 三码，例如 `I-A-S`。若并列，保留并列并在报告中说明不确定性。

### Big Five

反向题按 `6 - 原始分` 计入。

- 外向性 = B01 + reverse(B02) + reverse(B03) + B04
- 宜人性 = B05 + reverse(B06) + B07 + reverse(B08)
- 尽责性 = B09 + B10 + B11 + reverse(B12)
- 神经质 = B13 + reverse(B14) + B15 + reverse(B16)
- 开放性 = B17 + reverse(B18) + reverse(B19) + reverse(B20)

每项范围 4–20。建议解释：

- 4–9：相对较低
- 10–15：中等
- 16–20：相对较高

不要把分数解释成优劣。它只表示更可能适应某类学习/工作环境。

## 职业方向解释原则

Agent 生成职业方向时必须同时考虑：

1. RIASEC 前三码：决定“更可能被什么工作内容吸引”。
2. Big Five：决定“更可能适应什么工作方式、压力、协作和结构”。
3. 用户真实经历：测试结果必须回到用户回答和生活证据验证。
4. 行业与专业研究：职业方向必须再经过公开来源核验，不能直接从测试跳到专业结论。

输出职业方向时使用“假设”措辞，例如：“可能值得深入研究：数据分析 / 医学检验 / 教育咨询”。不要写“你最适合”“你一定应该”。
