# 高考志愿填报信息表

> 用途：收集生成志愿参考报告所需的高考信息。请在“你的填写”列填写。填完后保存文档，并把文件发回、粘贴文件内容，或授权 Agent 读取该文件。不要在终端里逐项回复。

## 填写原则

- 不确定的项目可以写“不确定”，Agent 会标记为待补充。
- 高考成绩和全省位次至少填一项；如果都有，优先使用位次。
- 所有招生规则、考试模式、选科要求和投档数据最终都必须回到省教育考试院或高校官方来源核验。
- 本表不是正式志愿表，只是生成参考报告的输入。
