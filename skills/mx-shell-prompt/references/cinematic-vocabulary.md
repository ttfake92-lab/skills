# 电影词汇库

## 摄影机

| 摄影机 | 特点 | 适用场景 |
|--------|------|---------|
| IMAX胶片摄影机 | 巨幅画面、极致清晰度、震撼沉浸感 | 史诗场景、自然风光、科幻大片 |
| ARRI Alexa 65 | 数字电影标杆、肤色还原优秀、宽容度高 | 剧情片、商业广告、人物特写 |
| RED V-Raptor | 8K超高分辨率、高帧率 | 需要后期裁切的画面、动作场景 |
| Sony Venice | 双原生ISO、低光出色 | 夜景、室内弱光 |
| Blackmagic URSA | 电影感强、性价比高 | 独立电影风格 |

## 镜头

| 镜头 | 特点 | 适用场景 |
|------|------|---------|
| Panavision C系列 | 经典变形宽荧幕镜头、柔和光晕 | 复古电影感、宽荧幕构图 |
| Cooke S4 | 温暖肤色、柔和散景 | 人物特写、浪漫场景 |
| Zeiss Master Prime | 极致锐利、低畸变 | 科幻、建筑、需要清晰度的场景 |
| Vintage Anamorphic | 强烈变形效果、光斑拉伸 | 复古风格、艺术感强的画面 |
| Canon K-35 | 柔和、轻微色散 | 文艺片、梦幻场景 |

## 胶片/数字质感

| 质感 | 描述 |
|------|------|
| Kodak Vision3 500T | 钨丝灯胶片、颗粒细腻、色彩饱和、经典好莱坞感 |
| Fuji Eterna | 色彩淡雅、偏冷调、日系电影常用 |
| 35mm胶片 | 中等颗粒、经典影院感 |
| 65mm/70mm胶片 | 极细颗粒、IMAX级别质感 |
| 胶片颗粒质感 | 添加数字胶片噪点，增加复古感和有机质感 |
| 低饱和度复古滤镜 | 压低色彩饱和度，营造怀旧氛围 |

| 霓虹紫+青绿 | 霓虹灯映照、夜城、赛博朋克 |
| 金色夕阳+琥珀 | 温暖、怀旧、浪漫 |
| 低饱和+青灰调 | 冷淡、孤独、文艺 |

| 黑白+高对比 | 黑色电影、经典、戏剧性 |
| 棕褐调+暖光 | 西部片、复古、粗犷 |

## 灯光

| 灯光 | 描述 | 情绪 |
|------|------|------|
| 自然侧光 | 窗户透入的柔和侧光 | 真实、宁静 |
| 伦勃朗光 | 面部三角形亮区 | 古典、戏剧性 |
| 明暗对比法 | 强烈明暗反差 | 神秘、紧张 |
| 霓虹环境光 | 多色霓虹灯映照 | 科幻、都市 |
| 金色逆光 | 夕阳/黄金时段逆光 | 浪漫、壮丽 |
| 阴天柔光 | 均匀无影的散射光 | 平静、忧郁 |
| 顶光 | 正上方照射 | 压迫、审判感 |
| 底光 | 正下方照射 | 恐怖、不安 |

## 视觉效果

| 效果 | 描述 |
|------|------|
| 胶片颗粒 | 添加有机噪点 |
| 镜头光晕 | 光源处的光晕扩散 |
| 色差/色散 | 边缘色彩偏移 |
| 光晕溢出(Halation) | 高光区域柔和溢出 |
| 动态模糊 | 运动中的模糊效果 |
| 浅景深 | 主体清晰、背景虚化 |
| 变形宽荧幕光斑 | 椭圆形光斑，变形镜头特征 |
