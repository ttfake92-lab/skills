# 赛博原子风格 — 完整示例

## 【基础设定】

机器人清道夫：{{Portrait 1}} 身形修长的人形机器人，1960年代原子朋克风格设计。拥有自主意识，面部led显示屏代替五官，显示马像素风格的表情（表情无动态效果，保持静帧显示）。身穿美式西部牛仔复古风格的生棕色牛仔帽、黑色哑光高腰设计皮衣夹克、黑色哑光皮手套、牛仔腰带和枪套。

非洲鸵鸟：{{Portrait 2}} 成年非洲公鸵鸟，样貌看起来呆傻，双眼不聚焦，歪吐着舌头。

场景：{{Portrait 6}} 末日爆发丧尸危机，大战过后的滨海半山度假别墅，1960年代原子朋克建筑风格，美式复古风。窗外正午刺眼阳光漫射，海面波光粼粼，烈日下空气微微热浪扭曲。室内为装修豪华，但已破败不堪的别墅休闲区，场景四处均有丧尸的尸体、撕裂的四肢、杂乱的生活用品、钞票、酒瓶酒杯以及血渍，仿佛经历过一场战斗的死寂氛围。整体氛围奢华松弛，兼具度假慵懒感与复古未来主义的高级质感，休闲的惬意的环境与丧尸的尸体形成极大的反差对比

声音：不需要配乐，仅保留同期声。

## 【氛围与画质】

风格核心：原子朋克、末日丧尸、电影级质感、超写实、极致逼真、真人实景拍摄、杜绝游戏CG感。

视觉基调：变形宽荧幕电影质感，使用IMAX胶片摄影机、搭配Panavision C系列镜头（添加动态模糊）拍摄。

色彩与影调：60年代复古科幻原子朋克美学，复古暖橙+海盐蓝高对比色调，胶片颗粒质感，复古广角镜头，低饱和度复古胶片滤镜，采用自然光照明，大面积落地窗透进柔和的自然侧光，在水磨石地面形成自然柔和的光影；整体光线通透均匀，高光不过曝，暗部保留完整细节，明暗过渡自然，营造寂静的光影氛围，带轻微复古胶片柔光质感。

视觉基调：参考美剧《辐射》美学风格、皮克斯动画人物动作控制，原本闲适的度假场景与恐怖丧尸形成强烈荒诞反差，整体保持窒息、孤寂的末日孤胆英雄既视感。

## 【画面内容】

分镜一：
景别：中景，拍摄机器人腿部以上位置。
构图：机器人背影作为前景居中位于画面中心位置；背景左侧为落地窗景；背景右侧为吧台。
运镜手法：跟随运动拍摄。
画面内容：机器人背对镜头向前方移动，手舞足蹈（边走边跳舞）地进入别墅休闲区，来到中心区域后，以一个潇洒的转身作为舞蹈EDN动作顶住，身体面对镜头静止片刻，led显示器保持 {{Portrait 5}} 绿色微笑表情。片刻后，画外传来玻璃酒瓶被打碎的声音，机器人受到惊吓，下意识地向画面左侧后撤半步，身体重心下沉，手臂微微蜷缩起来，led显示器瞬间切换至 {{Portrait 3}} 黄色害怕表情，视线看向画面右侧。

分镜二：
景别：全景，平视拍摄吧台全貌。
构图：空镜，平视拍摄吧台全貌，画面下方三分之一处为吧台和吧台椅子，上方为墙面酒柜。
运镜手法：镜头缓慢前推，渲染氛围。
画面内容：吧台的后方传来悉悉簌簌的声音，像是有什么东西躲在吧台下方翻找东西。

分镜三：
景别：特写，拍摄机器人正面面部表情。
构图：对称居中构图。
运镜手法：镜头缓慢前推，渲染氛围。
画面内容：机器人变得胆小，双手缓缓抬至下巴前方的位置呈防御姿态，全身忍不住地在发抖，led显示器保持 {{Portrait 3}} 黄色害怕表情。

分镜四：
景别：长焦，过肩镜头。
构图：机器人作为远景处于画面左侧位置，前景为鸵鸟后脑位置处于画面右侧，焦点对聚焦于机器人，产生浅景深的透视关系。
运镜手法：固定机位拍摄。
画面内容：鸵鸟背对镜头迅速起身，后脑勺从画面下方入画窜出，探出脑袋观察着机器人。机器人被突然出现的鸵鸟再次吓到，双手抱头不敢直视。

分镜五：
景别：全景，拍摄整个室内空间。
构图：平视拍摄，对称构图。画面左侧前景为黄色沙发，中景为机器人，背景为落地窗外的大海；画面右侧前景为桌面绿植，中景依次为吧台侧面、鸵鸟、酒柜背景墙，背景为混凝土支柱。
运镜手法：固定机位拍摄。
画面内容：机器人面部led显示器保持 {{Portrait 3}} 黄色害怕表情，双手缓缓离开头部，慢慢站直身体，视线看向画面右侧的吧台方向的鸵鸟。

分镜六：
景别：长焦特写，拍摄鸵鸟正面面部表情。
构图：居中构图。
运镜手法：固定机位拍摄。
画面内容：呆傻的鸵鸟歪吐着舌头，略微歪头，注视着镜头方向，观察着眼前的这个机器人。

分镜七：
景别：长焦特写，拍摄机器人侧面45度
构图：居中构图。
运镜手法：固定机位拍摄。
画面内容：机器人注视着画面右侧方向，面部表情由 {{Portrait 3}} 黄色害怕切换至 {{Portrait 4}} 白色思考。
