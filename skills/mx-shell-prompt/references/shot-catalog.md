# 景别 / 构图 / 运镜速查表

## 景别（Shot Types）

| 中文 | 英文 | 描述 | 情绪效果 |
|------|------|------|---------|
| 大特写 | Extreme Close-up (ECU) | 面部局部（眼、嘴）或物体细节 | 极度亲密、紧张、揭示细节 |
| 特写 | Close-up (CU) | 面部或手部 | 情感表达、人物内心 |
| 中近景 | Medium Close-up (MCU) | 胸部以上 | 对话、情感交流 |
| 中景 | Medium Shot (MS) | 腰部以上 | 日常互动、叙事推进 |
| 中远景 | Medium Wide (MW) | 膝盖以上 | 展示动作、环境关系 |
| 全景 | Wide Shot (WS) | 全身 | 展示人物与环境关系 |
| 远景 | Extreme Wide (EWS) | 人物在环境中很小 | 环境主导、孤独、渺小 |
| 建立镜头 | Establishing Shot | 场景全貌 | 交代地点、建立氛围 |
| 过肩镜头 | Over-the-shoulder (OTS) | 从一人肩后拍另一人 | 对话、窥视感 |
| 主观镜头 | POV | 角色视角 | 沉浸、代入感 |
| 插入镜头 | Insert | 物体细节 | 交代信息、制造悬念 |

## 构图（Composition）

| 中文 | 英文 | 描述 | 适用场景 |
|------|------|------|---------|
| 居中构图 | Centered | 主体置于画面正中 | 庄重、对峙、强调主体 |
| 三分法 | Rule of Thirds | 主体置于三分线交点 | 自然、平衡、经典 |
| 对称构图 | Symmetry | 画面左右/上下对称 | 秩序感、韦斯·安德森风格 |
| 画中画 | Frame within Frame | 利用门框、窗户等框住主体 | 窥视、聚焦、层次感 |
| 引导线 | Leading Lines | 利用线条引导视线至主体 | 引导、方向感 |
| 留白 | Negative Space | 大面积空白区域 | 孤独、思考、极简 |
| 前景遮挡 | Foreground Framing | 前景物体部分遮挡画面 | 层次、深度、窥视感 |
| 对角线构图 | Diagonal | 主体或线条沿对角线分布 | 动感、不安、张力 |
| 黄金比例 | Golden Ratio | 基于黄金分割的构图 | 和谐、自然美 |

## 运镜手法（Camera Movement）

| 中文 | 英文 | 描述 | 情绪效果 |
|------|------|------|---------|
| 固定机位 | Static/Lock-off | 相机不动 | 稳定、观察、客观 |
| 横摇 | Pan | 相机水平转动 | 跟随、揭示、环顾 |
| 俯仰 | Tilt | 相机垂直转动 | 仰望、俯视、揭示高度 |
| 推进 | Dolly In / Push In | 相机向主体移动 | 聚焦、紧张、亲密 |
| 拉远 | Dolly Out / Pull Back | 相机远离主体 | 揭示环境、疏离 |
| 跟踪 | Tracking | 相机跟随主体移动 | 同行、陪伴、沉浸 |
| 摇臂/升降 | Crane / Jib | 相机垂直升降 | 史诗感、上帝视角、揭示 |
| 手持 | Handheld | 手持摄影、轻微晃动 | 真实、紧张、纪录片感 |
| 斯坦尼康 | Steadicam | 平稳移动 | 流畅、梦境、跟随 |
| 甩镜 | Whip Pan | 快速横摇 | 转场、能量、紧张 |
| 变焦 | Zoom | 镜头焦距变化 | 聚焦/疏离、希区柯克变焦 |
| 移焦 | Rack Focus | 焦点在不同主体间切换 | 引导注意力、揭示关系 |
| 环绕 | Orbital | 相机环绕主体旋转 | 沉浸、仪式感、强调 |
| 缓慢前推 | Slow Push | 缓慢向主体推进 | 悬念、氛围渲染 |
| 一镜到底 | One-take / Long Take | 无剪辑连续拍摄 | 沉浸、真实、技术炫技 |
