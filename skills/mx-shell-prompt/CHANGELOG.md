# Changelog

本 Skill 遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [2.0.0] — 2026-06-05

### 重大变更：从「扩写一条 prompt」重构为「脚本 → 自动板块化 → 结构化提示词」

这是一次工作流重构，输出结构与触发场景都有变化，故升主版本号。

#### 新增

- **脚本自动分段（板块化）**：输入一整段脚本 / 故事 / 口播稿，自动按「场景切换 / 时间跳跃 / 情绪节拍 / 视角切换」拆成板块，每个板块 = 一次视频生成单元。展开前先给「分段概览」让用户确认。
- **「道具」独立板块**：关键物件单列成栏，强制交代状态与角色关系——这是画面连贯性最容易丢失的环节。
- **分镜 / 长镜头自动判断**：根据板块内容（动作密度、情绪转折）自动选择拆分镜还是走一镜到底，长镜头按「起 / 承 / 转」分阶段写。
- **时长 → 镜头数映射**：5s / 10s / 15s / 30s 对应的镜头数量参考。
- **画幅比例**：在「氛围与画质」中显式声明（16:9 / 9:16 / 2.39:1 等）。
- **跨板块一致性规则**：角色 / 风格 / 画幅在板块一定义，后续板块只写变化。

#### 变更

- 输出结构由 3 区块扩展为 **6 区块**：基础设定（角色）/ 道具 / 场景 / 声音 / 氛围与画质 / 画面内容（原先角色、场景、声音合并在「基础设定」内）。
- **定位收窄**：从宣称「适用所有文生视频工具」改为聚焦结构化分镜类工具（Seedance 2.0 / Kling / 即梦）。本格式（中文结构化块 + `{{Portrait N}}` 图片绑定）本就是这几家的地道格式；Sora / Runway / Pika 更适合英文散文式 prompt，SKILL.md 中已说明差异与改写方法。
- `template.md` 改为 6 区块 + 多板块骨架 + 长镜头变体。
- `example-cyber-atom.md` 套入板块结构、补充「道具」板块、补画幅。

#### 修复

- 修正示例中「视觉基调」字段重复出现的问题（第二处实为导演 / 风格参考，已改名为「风格参考」）。

## [1.0.0] — 2026-05

- 首个版本：把粗略想法扩写成结构化电影级视频提示词，三根支柱（谁在哪 / 什么感觉 / 发生什么），含景别·构图·运镜速查、电影词汇库、结构模板与完整示例。
