# MX Shell Prompt — 视频提示词写作

把粗略想法转化为结构化的电影级视频提示词，适用于 Seedance 2.0、Kling、Sora、Runway、Pika 等所有文生视频工具。

核心理念：每个好提示词由三根支柱构成 —— **谁在哪**（基础设定）、**什么感觉**（氛围与画质）、**发生什么**（画面内容）。

## 工作流程

### Step 1: 收集想法

从用户描述中提取：角色、场景、动作、情绪、风格偏好。用户的输入可以是一句话、一个故事梗概、一种情绪，甚至只是「我想要一个酷的赛博朋克场景」。

如果信息不足以写出高质量提示词，问 2-3 个关键问题：
- 单镜头还是多镜头分镜？（默认推荐 3-5 个镜头的多镜头分镜）
- 有特定视觉风格偏好吗？
- 目标时长？（5s / 10s / 15s / 30s）

用户如果说「直接写吧」，自行做出合理选择继续。

### Step 2: 构建基础设定

阅读 `references/template.md` 了解结构骨架。

**角色**：每个角色需要 —— 外貌体型、服装、标志性特征、气质/性格。如果用户提供了参考图，用 `{{Portrait N}}` 标记关联。

**场景**：环境类型、时间、天气、关键道具、空间布局、氛围细节。场景描述要具体到能「闻到」—— 不是「一个房间」，而是「午后阳光透过百叶窗洒进来的书房，灰尘在光柱中漂浮」。

**声音**：默认写「不需要配乐，仅保留同期声」。用户明确要求音乐或氛围音时再调整。

### Step 3: 构建氛围与画质

阅读 `references/cinematic-vocabulary.md` 选择合适的词汇。

**风格核心**：1-2 个艺术风格关键词 + 质量约束（电影级质感、超写实、真人实景拍摄、杜绝游戏CG感）。

**视觉基调**：指定摄影机 + 镜头 + 附加效果。不要只写「电影质感」，要写具体设备 —— 如「变形宽荧幕电影质感，使用IMAX胶片摄影机、搭配Panavision C系列镜头（添加动态模糊）拍摄」。

**色彩与影调**：描述具体色彩方案（不是「温暖」而是「复古暖橙+海盐蓝高对比色调」）、胶片质感、光线来源和方向、明暗关系、整体氛围。

### Step 4: 构建画面内容

阅读 `references/shot-catalog.md` 选择景别、构图和运镜。

为每个镜头写四个要素：
- **景别**：具体说明拍哪里（如「中景，拍摄机器人腿部以上位置」）
- **构图**：主体位置、前景/中景/背景层次、视觉重心
- **运镜手法**：固定、跟踪、推进、手持等
- **画面内容**：角色的具体动作、表情变化、事件发生的物理描述

多镜头分镜的设计原则：
- 镜头间景别要有变化（不要连续两个相同景别）
- 运镜节奏跟随情绪（紧张时推进，平静时固定）
- 要有一个「呼吸镜头」（全景或远景）让观众喘息
- 首尾镜头形成呼应或对比

### Step 5: 呈现与迭代

输出完整的 copy-paste-ready 提示词。然后问：「需要调整风格、增减镜头、修改运镜，或者调整某个具体镜头吗？」

## 写作原则

**视觉要超具体**：不是「漂亮的日落」，而是「金色夕阳的光线穿过百叶窗投下长影，皮肤上温暖的琥珀色调，画面边缘柔和的镜头光晕」。

**动作要有物理精度**：不是「角色很害怕」，而是「双手缓缓抬至下巴前方呈防御姿态，身体微微发抖，重心下沉」。

**必须有摄影语言**：每个镜头都包含景别 + 构图 + 运镜。这三者定义了电影感的 80%。

**禁止指令集中在风格核心**：把「禁止游戏CG感」「禁止动作僵硬」放在风格核心里，不要散落在每个镜头中。

**角色描述保持一致**：基础设定中写过的角色特征，分镜中不要重复描述，只写动作和表情变化。

**节奏设计**：紧张场景用短镜头快切，抒情场景用长镜头慢推。镜头时长要有变化，不要每个镜头一样长。

## 输出格式

```
【基础设定】
角色1：{{Portrait N}} [外貌、服装、特征、气质]
角色2：{{Portrait N}} [外貌、服装、特征、气质]

场景：[环境、时间、天气、道具、空间、氛围]

声音：不需要配乐，仅保留同期声。

【氛围与画质】
风格核心：[风格关键词]、电影级质感、超写实、极致逼真、真人实景拍摄、杜绝游戏CG感。
视觉基调：[摄影机] + [镜头]（[附加效果]）拍摄。
色彩与影调：[色彩方案]，[胶片质感]，[光线描述]，[明暗关系]，[整体氛围]。

【画面内容】
分镜一：
景别：[景别，具体说明拍哪里]
构图：[构图描述]
运镜手法：[运镜]
画面内容：[具体内容]

分镜二：
景别：
构图：
运镜手法：
画面内容：

[继续...]
```

## 常见错误

- **太模糊**：「电影质感」不指定摄影机和镜头 → 写具体设备型号
- **忘声音**：每个提示词都需要声音行 → 默认「仅保留同期声」
- **角色描述不一致**：分镜中重复基础设定已有的描述 → 只写新动作
- **运镜与情绪不匹配**：温柔场景用甩镜 → 运镜要服务情绪
- **风格堆砌**：同时引用 5 个风格 → 选 1-2 个强风格
- **没有景别和构图**：只写画面内容 → 每个镜头必须有景别+构图+运镜
- **负面指令散落**：每个镜头都写「不要CG感」 → 集中在风格核心

## 情绪 → 电影术语速查

| 情绪/场景 | 推荐术语 |
|----------|---------|
| 害怕/恐惧 | 浅景深、手持晃动、快速推进、冷色调 |
| 惊讶/揭示 | 缓慢前推、移焦（从前景到主体）、停顿 |
| 动作/追逐 | 甩镜、速度变化、低角度跟踪、高对比 |
| 浪漫/温柔 | 柔焦、暖色调、缓慢推轨、特写 |
| 紧张/悬疑 | 固定全景、沉默、缓慢变焦、暗调 |
| 孤独/沉思 | 远景留白、固定机位、自然光、低饱和 |
| 欢乐/活力 | 手持跟拍、高饱和、明亮光线、中景 |
| 史诗/壮丽 | 摇臂升降、远景、IMAX画幅、金色光线 |
